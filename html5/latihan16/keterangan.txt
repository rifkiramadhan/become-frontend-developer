---- Keterangan Latihan 16 ----

1. Pengertian
   - <html lang="en">
     - Artinya menggunakan bahasa english / inggris untuk website kita
   - <meta charset="UTF-8">
     - Artinya karakter website kita, jadi ketika membuat document baru maka script HTML nya seperti template utama kita
   - <meta name="viewport"...
     - Berfungsi untuk ketika website kita dibuka pada tampilan device atau seperti HP dan Tablet maka ukurannya akan
       responsive, jadi ini berdasarkan dari device apa yang digunakan
   - <title>
     - Berfungsi untuk membuat judul pada tab website kita
   - <head> (Header)
     - Header adalah judul halaman pada content website yang kita buat, dan header memiliki 6 versi
       - <h1>
       - <h2>
       - <h3>
       - <h4>
       - <h5>
       - <h6>
     - Jadi semakin besar head nya maka semakin kecil ukurannya, dan jarang sekali orang menggunakan h5 dan h6, paling
       sedikit orang menggunakan yaitu h4
     - Header ini sangat berfungsi sekali untuk mempresentasikan suatu halaman yang dibuat itu fungsinya untuk apa,
       jadi kita perlu menyesuaikan tag head nya pada setiap content sehingga relevan ketika dibaca oleh pengguna
   - <p> (Paragraf)
     - Adalah tag html yang berfungsi untuk membuat teks paragraf pada setiap halaman webste
   - <br>
     - Adalah tag html yang berfungsi untuk membuat line break pada halaman website, sehingga teks yang ada di posisi
       setelah tag penutup br akan jatuh kebawah
     - Dari segi desain UI lebih baik menggunakan satu buah tag br pada tiap paragraf agar space nya tidak terlalu jauh
   - <ol>
     - Ini adalah order list
   - <ul>
     - ini adalah unordered ist, artinya kita tidak menggunakan angka
   - <li>
     - Ini adalah list item
   - <a> (Achor)
     - Berfungsi untuk membuat link pada halaman website yang bertujuan untuk berpindah dari halaman satu ke halaman lain
     - target="_blank"
       - Berfungsi untuk membuka tab baru pada browser ketika tag link di tekan, agar tidak menimpa website yang dibuka
   - <img> (Images)
     - Berfungsi untuk membuat sebuah gambar pada halaman website kita
     - Jadi gambar itu sangat penting ketika kita ingin mempresentasikan desain yang lebih menarik lagi
     - alt="Gambar 1"
       - Adalah alternatif gambar yang akan muncul ketika gambarnya tidak terdeteksi
   - <section>
     - Section adalah suatu bagian pada content website, jadi kita ingin membuat susunan website itu lebih rapih lagi
       menggunakan section, jadi semuanya kita dapat kategorikan dengan berbeda-beda section nya
     - Ketika kita menggunakan section, pastikan kita memiliki h1 sebagai titlenya, itu adalah syarat dari section
     - Section dapat disatukan dan juga dapat dibedakan, tetapi lebih baik dibedakan saja agar memberikan dampak yang
       baik untuk SEO
   - <nav> (Navigation)
     - Berfungsi untuk membuat sebuah menu pada halaman website
   - <footer>
     - Adalah tempat untuk meletakkan suatu copyright sehingga nanti pengunjung yang datang itu dapat melihat, bahwa
       ini adalah perusahaan dari mana, jadi dapat memberikan effect branding kepada website kita sendiri, dan ini 
       cukup penting juga ketika kita sedang membangun website tersebut pada tahun berapa itu dapat dilihat juga, dan
       ini juga dapat meningkatkan dari segi kepercayaan pada costomer kita
   - <div>
     - Berfungsi untuk memudahkan kita ketika kita ingin mengcopy, mempaste atau memperbaiki nantinya ketika kita sedang
       mendevelop suatu website
     - Jadi div ini berfungsi seperti section tapi div ini ada di dalam section, kalau section seperti perbagian, tapi
       kalau div ini perbagian di dalam perbagian jadi lebih rapih lagis
   - <table>
     - Berfungsi untuk membuat table pada website, jadi datanya dapat terstruktur dengan baik
     - <tr> (Table Row / Baris)
       - Berfungsi untuk membuat baris table pada website
       - <th> (Table Header)
         - Berfungsi untuk membuat judul pada setiap baris table pada website
         - Akan memberitahu kepada browser bahwa saat ini kita menulis table header, sehingga tulisannya itu menjadi
           bold
       - <td> (Table Data / Anak dari th)
         - Berfungsi untuk membuat data dari setiap baris dan kolom table pada website
     - Jadi ketika kita ingin membuat sebuah desain yang merepresentasikan suatu informasi seperti comparison atau
       perbandingan maka table ini sangat cocok untuk digunakan
   - <form>
     - Berfungsi untuk menentukan character encodings yang diterima oleh server sebelum form dikirim
   - <button>
     - Berfungsi untuk ketika kita menuliskan username dan password lalu kita bisa mengirim value dan mengkosongkan value
   - Progress Bar
     - Biasa digunakan ketika kita ingin memberikan experience atau pengalaman kepada pengguna sehingga pengguna tersebut
       tahu pasti saat ini mereka berada di dalam proses apa
     - Sangat penting sekali untuk memberikan informasi, jadi kita sebagai pengguna itu tidak akan bingung untuk memberikan
       suatu indikator tertentu dan ini membuat tampilan website kita lebih cantik lagi
   - Color (Styling Tag HTML)
     - Berfungsi untuk memberikan pewarnaan pada halaman website agar terlihat lebih menarik
     - Code Hex: https://flatuicolors.com/
