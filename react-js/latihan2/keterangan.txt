---- Keterangan Latihan 2 ----

1. Pengertian
   - React
     - Kita akan menggunakan React JS sepenuhnya pada website agar lebih cepat, responsive, performance tinggi dll seperti 
       membuat lebih interaktif
   - App.js
     - Ini adalah tempat dimana kita akan melakukan proses coding dalam membangun single page application (SPA)
   - JSX 
    - Adalah (JavaScript XML) yang dimana kita memiliki variable JavaScript dengan di dalam nya itu kita sisipkan sebuah HTML
    - Dengan menggunakan JSX kita dapat menambahkan sintaks HTML di dalam suatu variable, jika bukan JSX dia hanya string biasa
